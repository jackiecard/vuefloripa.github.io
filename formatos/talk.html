# Talk

Esse é o formato mais tradicional do Meetup. Consiste de uma, ou mais, pessoas apresentando um tema, geralmente com slides, live coding ou algo do tipo.

As talks são submetidas no repositório do meetup como comentário na Issue correspondente a edição do encontro. As talks com o maior número de 👍 são selecionadas. Lembrando que dependendo da edição o número de talks escolhidas pode variar.
